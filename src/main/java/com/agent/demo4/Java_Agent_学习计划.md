# Java Agent 学习计划

## 第一阶段：基础入门（1周）
- 理解 Java Agent 概念与用途
- 学习 premain 与 agentmain 方法
- 编写第一个简单的 Java Agent

## 第二阶段：字节码操作（2周）
- 学习 ASM 框架
- 掌握 ClassVisitor、MethodVisitor
- 实现方法耗时统计 Agent

## 第三阶段：高级应用（1周）
- 学习 Instrumentation API
- 实现热替换与监控
- 了解 Attach API

## 第四阶段：实战项目（1周）
- 开发一个完整的性能监控 Agent
- 集成日志与告警
- 编写测试用例
