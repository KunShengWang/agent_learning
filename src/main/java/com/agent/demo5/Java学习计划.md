# Java 学习计划

## 📅 总体时间规划（建议 3~6 个月）

| 阶段 | 时间 | 学习内容 | 目标 |
|------|------|----------|------|
| 第一阶段 | 第1~2周 | Java 基础语法 | 掌握基本编程能力 |
| 第二阶段 | 第3~4周 | 面向对象编程 | 理解 OOP 思想 |
| 第三阶段 | 第5~6周 | 常用 API 与集合框架 | 熟练使用 Java 核心类库 |
| 第四阶段 | 第7~8周 | IO 流与多线程 | 掌握文件操作与并发编程 |
| 第五阶段 | 第9~10周 | 网络编程与数据库 | 理解网络通信与 JDBC |
| 第六阶段 | 第11~12周 | 项目实战 | 综合运用所学知识 |

---

## 📘 第一阶段：Java 基础语法（第1~2周）

- [ ] Java 发展历史与环境搭建（JDK、IDE）
- [ ] 基本数据类型与变量
- [ ] 运算符与表达式
- [ ] 流程控制语句（if、switch、for、while）
- [ ] 数组的定义与使用
- [ ] 方法定义与重载

## 📗 第二阶段：面向对象编程（第3~4周）

- [ ] 类与对象
- [ ] 封装、继承、多态
- [ ] 抽象类与接口
- [ ] static 与 final 关键字
- [ ] 内部类与匿名类
- [ ] 异常处理机制

## 📙 第三阶段：常用 API 与集合框架（第5~6周）

- [ ] String、StringBuilder、StringBuffer
- [ ] 包装类与自动装箱/拆箱
- [ ] 集合框架：List、Set、Map
- [ ] 泛型
- [ ] Collections 工具类
- [ ] 日期时间 API（Date、Calendar、LocalDateTime）

## 📕 第四阶段：IO 流与多线程（第7~8周）

- [ ] File 类与文件操作
- [ ] 字节流与字符流
- [ ] 缓冲流、转换流、对象流
- [ ] 多线程基础（Thread、Runnable）
- [ ] 线程同步与锁
- [ ] 线程池

## 📒 第五阶段：网络编程与数据库（第9~10周）

- [ ] 网络编程基础（Socket、ServerSocket）
- [ ] URL 与 HTTP 请求
- [ ] MySQL 数据库基础
- [ ] JDBC 连接与操作数据库
- [ ] 事务管理

## 📚 第六阶段：项目实战（第11~12周）

- [ ] 学生管理系统（控制台版）
- [ ] 图书管理系统（GUI 版）
- [ ] 简易聊天室（网络版）
- [ ] 个人博客系统（Web 版，可选）

## 💡 学习建议

1. **多动手**：每学一个知识点，立刻写代码验证。
2. **记笔记**：用 Markdown 记录重点和易错点。
3. **做项目**：学完基础后尽快进入项目实战。
4. **坚持每天写代码**：哪怕只有 30 分钟。
5. **善用搜索引擎**：遇到问题先自己查资料解决。
