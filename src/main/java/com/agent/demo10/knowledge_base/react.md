# ReAct Agent

ReAct 是 Reasoning and Acting 的缩写，核心思想是让模型在推理和行动之间交替进行。

一个典型 ReAct 循环包括：

1. 观察当前任务和上下文。
2. 思考下一步应该做什么。
3. 如果需要外部信息或外部动作，就调用工具。
4. 读取工具结果。
5. 决定继续调用工具，还是输出最终答案。

ReAct 的关键不是某一个固定 prompt，而是让模型拥有“观察、行动、再观察”的闭环能力。

在代码实现上，ReAct Agent 往往需要保存 messages，因为 messages 里包含用户目标、模型历史回复、工具调用参数和工具执行结果。
